# Configurando MLFLow como container

## Ambiente

- debian12 (bookworm)
- Docker

## Configuração

1) criar container MLFlow

    a) na pasta mlflow, execute ./run.sh

    b) ou, ./build.sh e ./create.sh

    c) use ./destroy p/ remover o conteiner


## Execução

1) Abrir Tracking Server (mlflow ui) em:

    > http://127.0.0.1:5000
