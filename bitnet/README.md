# Kolmogorov-Arnold Network (KAN)

## Ambiente

- debian12 (bookworm)
- miniconda
- vscode (IDE) + jupyter notebook (extensão)


## Configuração
```
conda env create -f environment.yml
conda activate bitnet
pip install -r requirements.txt
```

## Execução

1) bitlinear.ipynb
